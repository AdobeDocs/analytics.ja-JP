# セグメント

セグメントとは、データを独自に絞り込んだサブセット、つまり指定した規則でデータをフィルタリングしたものです。

セグメントの基準は次のとおりです。

* ヒット数
* 訪問者
* 訪問回数

コンポーネ [ントユーザーガイドの](/help/components/c-segmentation/seg-home.md) 「Analyticsのセグメント化」を参照してください。
