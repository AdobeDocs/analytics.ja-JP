# 指標

指標は、ビュー数、クリックスルー数、リロード数、平均滞在時間、数量、注文件数、売上高など、訪問者の行動に関する量的な情報です。

指標はレポートの基盤であり、データの関係性を確認して把握する助けになります。これにより、Web サイトに関する異なるデータセットを並べて比較することができます。
