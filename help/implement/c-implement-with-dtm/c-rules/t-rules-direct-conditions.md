# ダイレクト型ルールの条件の作成

ダイレクト型ルールの条件の作成。

1. **[!UICONTROL 条件]**&#x200B;ダイアログから、直接呼び出しで `_satellite.track()` に渡す文字列を（引用符なしで）指定します。

   ![](assets/conditions-direct-call.png)

   >[!NOTE]
   >
   >ダイレクト型ルール内の `_satellite.track()` に渡される文字列を UI から指定するときは、上の説明のとおり、値を引用符で囲まないでください。[カスタマイズのページコード](/help/implement/c-implement-with-dtm/c-aa-tool/customize-page-code.md)をエディターで挿入するときは引用符が必要です。
