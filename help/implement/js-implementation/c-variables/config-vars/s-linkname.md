# s.linkName

カスタムリンク、ダウンロードリンクまたは離脱リンクのレポートに表示される名前。 100 文字以下に切り捨てられます。どの文字列にも設定できます。これはイメージリクエストの `pev2` パラメーターです。

If set with `linkType`, an image request will be sent as a download, custom or exit link.
