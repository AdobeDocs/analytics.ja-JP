# getVisitStart

新しい訪問が開始しているかを確認します。

**設定変数**

なし

**パラメーター**

c = (文字列) トラッキングのための cookie の名前。

**戻り値**

（整数値）訪問の最初のページは 1、それ以外は 0。

**サンプル呼び出し**

```
s.eVar50 = s.getVisitStart("s_visit");
```
