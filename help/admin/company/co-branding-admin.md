# ブランド提携

ブランド提携画像の管理ページを使用すると、Reports &amp; Analytics のダウンロードしたレポートとレガシーダッシュボードに会社のロゴを表示できます。ブランド提携は、Analysis Workspace では使用されません。

最良の結果を得るには、次の画像のガイドラインを使用します。

* PDF 出力の場合は、417px x 125px の画像をアップロードすると最適に表示されます。
