# 新しいレポートスイート - 設定

新しいレポートスイートを作成するには、事前定義済みのテンプレートを選択するか、モデルとなる既存のレポートスイートを 1 つ選択して使用します。

使用される要素の説明。[レポートスイートの作成時](/help/admin/c-manage-report-suites/c-new-report-suite/t-create-a-report-suite.md)に使用されるエレメントの説明。
