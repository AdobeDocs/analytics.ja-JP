# ソースレポートスイートからコピーされない設定

既存のレポートスイートから設定をコピーした場合、システムのメッセージで、新しいレポートスイートにコピーされない設定が特定されます。

以下の設定は、テンプレートまたは重複したレポートスイートからコピーされません。
