# レポートスイートテンプレートの概要

レポートスイートテンプレートは、複数のタイプのレポートスイートに対して最も一般的な設定を行います。

新しいレポートスイートを作成する時間を節約するには、設計したレポートスイート設定に類似したテンプレートを選択します。レポートスイートテンプレートを選択しても、導入は何も制限されません。レポートスイートテンプレートによって適用されるすべての設定は、レポートスイートの作成後に変更できます。
