# Adobe Analytics はじめての管理ガイド

Adobe Analyticsでユーザーに役割を割り当てる前に、CX Enterpriseで最初の管理者としてユーザーを割り当てる必要があります。 その後、最初の管理者は、組織内のユーザーを他の主要な役割とプロビジョニングできます。

1人目の管理者は、Adobe Analyticsを含むCX Enterprise ソリューションを他の部門で利用できるようにするための出発点となります。

## 最初の管理者のプロビジョニングプロセス

契約書に署名した後：

1. アドビのプロビジョニングチームは、アカウントを作成するための準備をします。

1. 最初の管理者には、契約の開始日より前に、ログイン資格情報が記載された電子メールが送られます。

>[!IMPORTANT]
>
>   契約を結ぶ前に、最初の管理者の正確な連絡先情報がアドビに提供されていることを確認してください。

## 次の手順

最初の管理者が組織にプロビジョニングされた後、最初の管理者は組織内のユーザーを他の主要な役割とプロビジョニングできます。

Adobe Analyticsの主要な役割を使用して組織内のユーザーをプロビジョニングする方法について詳しくは、「[Adobe Analyticsの管理者の役割](/help/admin/admin-console/admin-roles-in-analytics.md)」を参照してください。
