# ユーザーと製品の管理

Admin Console で Analytics ユーザー、グループおよび製品を管理します。

>[!IMPORTANT]
>
>ユーザーおよび製品の管理は、 [Adobe Admin Console](https://helpx.adobe.com/jp/enterprise/using/admin-console.html). ユーザーを移行する時期は、アドビから通知されます。
