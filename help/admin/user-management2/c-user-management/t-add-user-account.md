# ユーザーアカウントの追加

>[!IMPORTANT]
>
>ユーザーおよび製品管理は、[Admin Console](https://helpx.adobe.com/jp/enterprise/using/admin-console.html) に移行されます。ユーザーを移行する時期は、アドビから通知されます。すべてのお客様が移行した後、**[!UICONTROL Analytics]**/**[!UICONTROL 管理者]**/**[!UICONTROL すべての管理者]**/**[!UICONTROL ユーザー管理]**&#x200B;のヘルプコンテンツは廃止されます。

ユーザーアカウントを追加する方法について手順を説明します。

1. **[!UICONTROL Analytics]** / **[!UICONTROL 管理者]** / **[!UICONTROL すべての管理者]** / **[!UICONTROL ユーザー管理]**。
1. 「**[!UICONTROL 新しいユーザーの追加]**」をクリックします。
1. [ユーザーアカウントの説明](/help/admin/user-management2/c-user-management/users.md#section_14A7E169514A42A88E06387CC7C2E9AD)で指定したフィールドに入力します。
