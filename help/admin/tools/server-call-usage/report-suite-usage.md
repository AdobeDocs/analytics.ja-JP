# レポートスイートの使用状況の表示

「レポートスイートの使用状況」タブには、請求会社に関連付けられたすべてのログイン会社のレポートスイートごとに、現在の使用期間のサーバー使用状況データが表示されます。

**[!UICONTROL Analytics]**／**[!UICONTROL 管理者]**／**[!UICONTROL サーバーコールの使用状況]**／**[!UICONTROL レポートスイートの使用状況]**

>[!IMPORTANT]
>
>レポートスイートがCX Enterprise組織にリンクされていない場合、その使用データはこのダッシュボードに反映されません。 また、請求IDは複数のCX Enterprise組織に関連付けることができます。組織と請求IDの間には必ずしも1:1の関係があるわけではありません。

レポートスイート使用状況ダッシュボード

* CX Enterprise組織内の各レポートスイートの現在の使用期間のサーバーコール使用状況（すべての呼び出し、プライマリ、セカンダリ、モバイルプライマリ、モバイルセカンダリ）を表示します。
* サーバー呼び出しカテゴリごとの全体的な使用率を表示します。
* 日次で更新されます。
* ダウンロード可能です。
* **[!UICONTROL アラート管理]** UI にアクセスできます。

![](/help/admin/tools/server-call-usage/assets/report-suite-usage.png)

## ダッシュボード設定 {#settings}

| 列 | 定義 |
|--- |--- |
| レポートスイート名 | レポートスイートのわかりやすい名前 |
| すべての呼び出し（合計の%） | 現在の使用期間に発生したすべてのサーバーコール。 |
| プライマリ呼び出し（%） | 現在の使用期間に発生したすべてのプライマリサーバーコール（および合計の割合）。 |
| セカンダリ呼び出し（%） | 現在の使用期間に発生したすべてのセカンダリサーバーコール（および合計の割合）。 |
| モバイルプライマリ（%） | 現在の使用期間に発生したすべてのモバイルプライマリサーバーコール（および合計の割合）。 |
| モバイルセカンダリ（%） | 現在の使用期間に発生したすべてのモバイルセカンダリサーバーコール（および合計の割合）。 |

{style="table-layout:auto"}

## 使用状況レポートのダウンロード {#download}

このオプションを使用すると、現在の使用状況データと、現在の使用期間より前の期間（2015年1月まで）のデータをダウンロードできます。 レポートは.csv ファイルとしてダウンロードされます。

1. 少なくとも1つのレポートスイートを選択してください。
1. 「**[!UICONTROL レポートをダウンロード]**」をクリックします。

   ![](/help/admin/tools/server-call-usage/assets/download_report.png)

| レポート要素 | 説明 |
|--- |--- |
| ファイル名 | ハードコード名：使用状況レポート`day and time of report creation.csv` |
| 含まれるレポートスイート | レポートサーバーの使用状況ページで選択したレポートスイートは、このリストに含まれます。 |
| 含まれるコールタイプ | すべての呼び出し（デフォルト）、プライマリ、セカンダリ、モバイルプライマリ、モバイルセカンダリの任意の組み合わせを指定します。 |
| 時間範囲 | 現在の使用期間を選択するか、カスタムの範囲を指定します。  カスタムの範囲の場合は、範囲の開始と範囲の終了を指定します。 <br>**メモ：**&#x200B;なお、2015年1月より前の使用状況データをダウンロードすることはできません。</br> |

{style="table-layout:auto"}

1. 「**[!UICONTROL ダウンロード]**」をクリックします。

ダウンロードした .csv ファイルのスクリーンショットを次に示します。 レポートスイート ID の列が含まれます。 レポートスイート ID は、英数字のみを含む一意の ID を指定します。 レポートスイートの作成後は、この ID を変更できなくなります。
