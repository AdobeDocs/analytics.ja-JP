# レポートスイートマネージャー

レポートスイートでは、次のいずれかのデータストアに関する完全な独立したレポートが定義されます。

* 単一の web サイト（最も一般的）

* 単一の web サイトの一部またはサブセット

* いくつかのサイトの数値の合計で構成されるグローバルセグメント

Adobe Analytics ソリューションでは、これらのデータストアを集計し、レポートを作成できます。 管理者レポートスイートマネージャーを使用すると、レポートスイートでのデータの処理方法を管理するルールを定義できます。

Adobe Analytics ソリューションにログインしたら、使用するレポートスイートを 1 つ選択します（レポートスイートを組み合わせたロールアップを使用する場合を除く）。

レポートスイートにアクセスするには：

**[!UICONTROL Analytics]**／**[!UICONTROL 管理者]**／**[!UICONTROL すべての管理者]**／**[!UICONTROL レポートスイート]**

>[!NOTE]
>
>仮想レポートスイートは、**[!UICONTROL Analytics]**／**[!UICONTROL コンポーネント]**／**[!UICONTROL 仮想レポートスイート]**&#x200B;で管理されます。 [仮想レポートスイートのドキュメント](/help/components/vrs/vrs-about.md)を参照してください。
