# レポートスイートテンプレートの概要

レポートスイートテンプレートには、複数の種類のレポートスイートを対象とした最も一般的な設定が定義されています。

新しいレポートスイートを作成する際の時間を節約するために、設計したレポートスイート設定に類似したテンプレートを選択できます。 レポートスイートテンプレートを選択しても、実装が制限されることはありません。 レポートスイートテンプレートによって作成された設定は、レポートスイートの作成後に変更できます。

[詳細情報](/help/admin/tools/manage-rs/rs-templates/default-rs-template.md)
