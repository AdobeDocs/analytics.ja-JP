# メディア管理

このインターフェイスを使用して、メディアの測定とレポートで使用する変数を予約できます。 これらの変数がeVar、prop、イベントと同様に動作しますが、変数制限に対してカウントされません。 有効にすると、Analytics メニューに新しいメディアレポートのセットが表示されます。

**[!UICONTROL Analytics]**／**[!UICONTROL 管理者]**／**[!UICONTROL レポートスイート]**／**[!UICONTROL 設定を編集]**／**[!UICONTROL メディア管理]**。

レポートスイートの編集時には、次のメディア管理オプションを使用できます。

* [!UICONTROL **メディア レポート**]

  これらのチェックボックスを使用して、ストリーミングメディアサービスに固有のディメンションと指標を有効にします。

   * [メディアコアディメンション](/help/components/dimensions/sm-core.md)
   * [Media core metrics](/help/components/metrics/sm-core.md)
   * [メディア広告のディメンション](/help/components/dimensions/sm-ads.md)
   * [メディア広告指標](/help/components/metrics/sm-ads.md)
   * [メディアチャプターディメンション](/help/components/dimensions/sm-chapters.md)
   * [メディアチャプター指標](/help/components/metrics/sm-chapters.md)
   * [メディア品質ディメンション](/help/components/dimensions/sm-quality.md)
   * [メディア品質指標](/help/components/metrics/sm-quality.md)
   * [ビデオメタデータディメンション](/help/components/dimensions/sm-video-metadata.md)
   * [動画メタデータ指標](/help/components/metrics/sm-video-metadata.md)
   * [オーディオメタデータディメンション](/help/components/dimensions/sm-audio-metadata.md)
   * [プレイヤーの状態トラッキング指標](/help/components/metrics/sm-player-state.md)

* [!UICONTROL **メディア分類**]

  ストリーミングメディアサービスディメンションに固有の分類は、[!UICONTROL  メディア分類] ページで利用できます。 これらは、コンバージョン変数で使用できる[ コンバージョン分類](/help/admin/tools/manage-rs/edit-settings/conversion-var-admin/conversion-classifications.md)と同様に動作します。 [!UICONTROL Media Reporting]で特定のコンポーネントを有効にすると、分類ディメンションが自動的に作成されます。 このインターフェイスを使用して、独自の分類ディメンションを作成したり、分類データをアップロードしたりできます。

Adobe Analyticsにストリーミングメディアデータを組み込む方法について詳しくは、[Adobe ストリーミングメディアサービスの概要](https://experienceleague.adobe.com/ja/docs/media-analytics/using/media-overview)を参照してください。
