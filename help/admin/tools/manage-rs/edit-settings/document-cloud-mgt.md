# Adobe Analytics での Document Cloud の設定

Document Cloud Analyticsを有効にすると、PDF固有のディメンションと指標をレポートスイートで使用できるようになります。

**[!UICONTROL Analytics]** > **[!UICONTROL 管理者]** > **[!UICONTROL レポートスイート]** > **`<select report suite>`** > **[!UICONTROL 設定を編集]** > **[!UICONTROL Document Cloud Management]**

レポートスイートを編集する際は、次のDocument Cloud管理オプションを使用できます。

* [!UICONTROL **Document Cloud レポート**]

  ストリーミングメディアサービスに固有のレポートは、メディアレポート ページで利用できます。 Media Reportingについて詳しくは、[Document Cloud レポートの設定](/help/admin/tools/manage-rs/edit-settings/document-cloud-config.md)を参照してください。

* [!UICONTROL **Document Cloud Classifications**]

  Document Cloudに固有の分類は、Document Cloudの分類ページで確認できます。 分類について詳しくは、[ コンバージョン分類](/help/admin/tools/manage-rs/edit-settings/conversion-var-admin/conversion-classifications.md)を参照してください。
