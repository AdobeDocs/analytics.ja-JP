# デフォルトの指標

[!UICONTROL **設定を編集**]／[!UICONTROL **個々のレポートスイートの設定**]／[!UICONTROL **デフォルトの指標**]

ユーザーが指標のカスタムセットを選択しない限り、Analyticsでは、すべてのレポートとプロジェクトにデフォルトの（すぐに使用できる）指標セットが表示されます。 選択した指標は、関連するレポートスイートのすべてのユーザーに表示されます。 一度に更新できるレポートスイートのデフォルト指標は1つだけです。 これらの設定は、デフォルトの指標が表示される順序を保証するものではありません。

詳しくは、次を参照してください。

* [Analysis Workspace の指標](/help/analyze/analysis-workspace/components/apply-create-metrics.md)

* [計算指標](/help/components/calculated-metrics/cm-overview.md)
