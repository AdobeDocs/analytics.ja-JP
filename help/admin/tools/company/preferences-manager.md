# 設定マネージャー

環境設定マネージャーページでは、Excelおよびコンマ区切り値（CSV）レポート出力のレンダリング方法を設定できます。

**[!UICONTROL Analytics]**／**[!UICONTROL 管理者]**／**[!UICONTROL すべての管理者]**／**[!UICONTROL カンパニー設定]**／**[!UICONTROL セキュリティ]**

* CSVに非推奨の日付形式を使用（8月30日、20XX）
* CSVにISO 8601日付形式を使用（20XX-08-30）
