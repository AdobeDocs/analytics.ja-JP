# データソース

データソース機能を使用すると、オフラインソースから Analytics にデータをインポートできます。 インポートしたデータは、ネイティブに収集されたデータと同様に扱うことができます。

データソースを使用して、以下を行います。

* オフラインデータとweb サイトから収集したデータを統合。
* web サーバー、コールセンター、その他のシステムからデータを収集します。 そして、それをCX Enterpriseにインポートし、組織のあらゆる手段からコンバージョンを確認します。
* SearchCenterを通じて、オフラインのリードフルフィルメントにもとづいて、キーワード入札管理を最適化します。

データソースがレポートスイートでアクティブ化されると、FTPの場所が提供されるので、レポートに統合するファイルをアップロードできます。 アップロードが完了すると、Adobeサーバーは自動的にデータを処理し、レポートスイートに統合します。

[ データソースのヘルプ ](/help/import/data-sources/overview.md)を参照してください。
