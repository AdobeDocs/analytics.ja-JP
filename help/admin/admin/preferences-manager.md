# 設定マネージャー

設定マネージャーページを使用すると、Excel およびコンマ区切り値 (CSV) レポート出力のレンダリング方法を設定できます。

**[!UICONTROL Analytics]**／**[!UICONTROL 管理者]**／**[!UICONTROL カンパニー設定]**／**[!UICONTROL 基本設定]**

* CSV に非推奨の日付形式を使用（Aug 30, 2013）
* CSV に ISO 8601 の日付形式を使用（2013-08-30）
