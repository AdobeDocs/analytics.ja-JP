# 処理ルール履歴の表示

処理ルールの変更は履歴で表示できます。

1. **[!UICONTROL Analytics]**／**[!UICONTROL 管理者]**／**[!UICONTROL レポートスイート]**&#x200B;の順にクリックします。
1. レポートスイートを選択します。
1. **[!UICONTROL 設定を編集]**／**[!UICONTROL 一般]**／**[!UICONTROL 処理ルール]**&#x200B;の順にクリックします。
1. **[!UICONTROL 処理ルール]**&#x200B;ページで、「**[!UICONTROL 履歴の表示]**」タブをクリックします。
