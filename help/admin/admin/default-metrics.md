# 標準指標

Analyticsでは、ユーザーがカスタム指標のセットを選択しない限り、すべてのレポートおよびプロジェクトに標準の指標セットが表示されます。選択した指標は、関連付けられたレポートスイートのすべてのユーザーに表示されます。一度に更新できるレポートスイートのデフォルトの指標は 1 つだけです。これらの設定により、デフォルトの指標の表示順が保証されるわけではありません。

詳しくは、   

*[Metrics in Analysis Workspace](/help/analyze/analysis-workspace/components/apply-create-metrics.md)
*[Calculated Metrics](/help/components/c-calcmetrics/cm-overview.md)
