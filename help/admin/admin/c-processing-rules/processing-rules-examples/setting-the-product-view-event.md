# 製品の概要ページからの「製品表示」イベントの設定

ページ価名や変数内の特定の値に基づいてイベントを設定できます。

| ルールセット | 値 |
|---|---|
| 条件 | ページ名が製品の概要と同じである場合 |
| アクション | 「イベントの設定」の「製品表示」イベントを「カスタム値」の「1」に設定する |
