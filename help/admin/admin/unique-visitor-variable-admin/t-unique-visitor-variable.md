# 個別訪問者変数の指定

訪問者の識別子を含んでいる eVar を指定する方法について手順を説明します。

この変数は、eVar を指定して個別訪問者の識別子を格納します。この変数を使用すると、独自の一意の識別子を使って顧客のアクティビティを報告できます。

1. **[!UICONTROL Analytics]**／**[!UICONTROL 管理者]**／**[!UICONTROL レポートスイート]**&#x200B;の順にクリックします。
1. レポートスイートを選択します。
1. **[!UICONTROL 設定の編集]**／**[!UICONTROL コンバージョン]**／**[!UICONTROL 実訪問者変数]**&#x200B;の順にクリックします。
1.  「**[!UICONTROL 利用可能な eVar]**」チェックボックスを選択してドロップダウンリストを有効にします。
1. ドロップダウンリストから eVar を選択し、「**[!UICONTROL 保存]**」をクリックします。
