# マーケティングチャネル

マーケティングチャネルは、サイトに訪問者がどのように（どのチャネルを経由して）到達するかを把握するために一般に使われます。追跡するチャネルと追跡する方法に応じて、マーケティングチャネル処理ルールを作成してカスタマイズできます。
