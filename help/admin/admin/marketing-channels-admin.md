# マーケティングチャネル

マーケティングチャネルは、一般に、どのように(どのチャネルを通じて)訪問者がサイトに到達するかに関するインサイトを提供するために使用されます。 追跡するチャネルと追跡する方法に応じて、マーケティングチャネル処理ルールを作成してカスタマイズできます。

詳しくは、[マーケティングチャネル](/help/components/c-marketing-channels/analyze-mc.md)を参照してください。
