# メニューのカスタマイズ

この機能を使用すると、Reports &amp; Analyticsでユーザーに表示されるレポートメニューをカスタマイズできます。 レポートの表示/非表示を切り替えたり、すべてのユーザーの異なるフォルダーに移動したりできます。 この機能は、組織が特定のレポートのみを使用し、左側のメニューを未使用または無関係なデータで混乱させない場合に特に便利です。

メニューの変更は、レポートスイートにアクセスするすべてのユーザーに適用されます。 ただし、ユーザーはレポートスイートの使用中にメニューのデフォルト設定を復元できます。

>[!NOTE]メニュー構造の変更は、Report Builder および Ad Hoc Analysis には影響しません。ただし、レポート名の変更は、両方のツールに反映されます。
