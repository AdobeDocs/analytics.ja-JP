# コンポーネントの説明を追加

Analytics 管理者は、レポートスイート内でコンポーネント (Dimension、指標、セグメントおよび日付範囲 ) の説明を追加できます。

組織内のユーザーは、Analysis Workspaceでプロジェクトを作成する際、またはレポートを表示する際に、説明を表示できます。 説明は、ユーザーが様々なコンポーネントの最適な使用方法を理解するのに役立ちます。
