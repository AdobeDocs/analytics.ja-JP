# 異常値検出をオフにする

異常値検出は、列レベルでオフにできます。

1. 列の ![ 設定 ](/help/assets/icons/Setting.svg) を選択します。
   ![](assets/turn-off-anomalies.png)

1. **[!UICONTROL 異常値を表示]** の選択を解除します。
