Adobe Analytics でのテンプレートの使用について詳しくは、次のリソースを参照してください。

* [テンプレートの使用](/help/analyze/analysis-workspace/templates/use-templates.md)

* [テンプレートの作成と管理](/help/analyze/analysis-workspace/templates/create-templates.md)
