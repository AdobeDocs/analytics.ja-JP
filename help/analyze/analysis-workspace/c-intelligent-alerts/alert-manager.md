# アラートの管理

アラートマネージャーでは、既存のアラートを管理できます。 タグ付け、名前の変更、削除など、アラートに対して様々な管理タスクを実行できます。

アラートマネージャーで既存のアラートを管理する方法については、[ アラートマネージャー ](/help/components/c-alerts/alert-manager.md) を参照してください。
