# ブレットグラフ

目的の値が、他のパフォーマンス範囲（目標）と照らし合わせて、どのように比較または測定されるかを確認できます。

![](assets/bullet-image.png)

ブレットグラフは、1 つ主要測定値（例えば、現在の年次累計売上高）を使用し、パフォーマンスの質的範囲を入力できます（例えば、ターゲット売上高と比較）。高、中、低を指定できます。「[!UICONTROL ビジュアライゼーション設定]」で目標範囲を指定できます。
