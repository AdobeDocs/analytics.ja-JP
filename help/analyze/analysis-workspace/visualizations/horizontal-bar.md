# 横棒グラフおよび積み重ね横棒グラフ

このビジュアライゼーションでは、1 つ以上の指標の様々な値を表す横棒グラフが表示されます。

![](assets/horizontal_bar.png)

## 積み重ね横棒グラフ {#section_50C08E9E20A94024A6553BC352ADB597}

![](assets/horizontal-bar-stacked.png)
