# Report Builder とは

Adobe Report Builder は Microsoft Excel のアドインです。これにより、カスタマイズしたAdobe Analyticsデータのリクエストを作成し、Excelワークシートに挿入できます。 リクエストはセルから動的に参照できます。さらに、Report Builder によるデータの表示形式は更新やカスタマイズが可能です。
