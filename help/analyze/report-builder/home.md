# Report Builderとは？

Adobe Report Builder は Microsoft Excel のアドインです。レポートおよび分析データからカスタマイズしたリクエストを作成し、Excelワークシートに挿入できます。 リクエストはセルから動的に参照できます。さらに、Report Builder によるデータの表示形式は更新やカスタマイズが可能です。
