# 開始日または終了日

開始日および終了日の略語です。

* cd-1d = 昨日
* cm-1d = 先月の最終日
* cw-1w = 先週の最初の日
* cw+1w-1d = 現在の週の最終日
* cd-2d = 2 日前
* cw-2w = 2 週間前の週の最初の日

未来の日付を設定しても、未来に関する値はレポートされないことに注意してください。
