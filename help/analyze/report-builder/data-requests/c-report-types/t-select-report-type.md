# レポートタイプの選択

データリクエストのベースレポートタイプを選択する手順です。

1. ウィンドウにナビゲ [!UICONTROL Request Wizard: Step 1] ートします。
1. Select a report suite from the **[!UICONTROL Report Suite]** menu.
1. データリクエストのベースレポートタイプをクリックして選択します。
