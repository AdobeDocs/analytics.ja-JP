# レポートスイートを選択します。

リクエストウィザードで表示されるメニューからレポートスイートを選択する手順です。

1. Excel で、「**[!UICONTROL アドイン]**」をクリックします。
1. ツールバーで、「**[!UICONTROL 作成]**」をクリックします。
1. [!UICONTROL リクエストウィザード：ステップ 1] ウィンドウの「[!UICONTROL レポートスイート]」でレポートスイートを選択します。
