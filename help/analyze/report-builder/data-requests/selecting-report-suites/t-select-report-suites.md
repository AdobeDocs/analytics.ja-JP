# レポートスイートを選択します。

リクエストウィザードで表示されるメニューからレポートスイートを選択する手順です。

1. In Excel, click **[!UICONTROL Add-Ins]**.
1. In the toolbar, click **[!UICONTROL Create]**.
1. ウィンドウ [!UICONTROL Request Wizard: Step 1] で、の下のレポートスイートを選択しま [!UICONTROL Report Suite]す。
