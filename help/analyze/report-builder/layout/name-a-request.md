# リクエストの名前の設定

リクエストに対してカスタム名を指定することができます。この名前はリクエストマネージャーに表示されます。

詳しくは、[表示ヘッダーのフォーマット](/help/analyze/report-builder/layout/t-format-display-headers.md)を参照してください。
