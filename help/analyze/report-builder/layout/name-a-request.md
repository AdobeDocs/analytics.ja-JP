# リクエストの名前の設定

リクエストに対してカスタム名を指定することができます。この名前はリクエストマネージャーに表示されます。

詳しくは、「表示ヘッダーの形式設 [定」を参照してください](/help/analyze/report-builder/layout/t-format-display-headers.md)。
