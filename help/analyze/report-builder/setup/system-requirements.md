# 必要システム構成

推奨システム構成および前提条件です。

* Windows XP 以降
* 32 ビットまたは 64 ビットの Microsoft Excel 2007 以降
* OS X はサポート対象外

**前提条件**

* 貴社の契約に Report Builder が含まれている。
* 使用するユーザーが、事前定義された Report Builder のユーザーグループに割り当てられている。[!DNL Analytics] 管理者も、このグループに属している必要があります。
* ユーザーが Windows に管理者権限を持つユーザーとしてログインしている。
