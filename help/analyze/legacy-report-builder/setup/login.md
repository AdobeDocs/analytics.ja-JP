# Report Builder へのログイン

{{legacy-arb}}

>[!IMPORTANT]
>
>Report Builder バージョン 5.6.47以降では、CX Enterprise ログインのみがサポートされ、Site Catalyst シングルサインオンや標準ログインなどの従来のログインはサポートされていません。 **2021年7月22日までに、すべてのReport Builder ユーザーはReport Builder アドインをバージョン 5.6.47以降に更新する必要があります。これには、ログインプロセスに関する重要な更新が含まれます。**

Report Builderにログインするには、CX Enterprise ログインアカウントを使用します。

## Adobe CX Enterpriseにログインします {#section_1FA230F35AB54021A874A7A28DE4C850}

CX Enterprise ログインでは、Enterprise ID（電子メールとパスワード）を使用してAdobe CX Enterpriseにログインできます。

**[!UICONTROL ログイン]**／**[!UICONTROL Enterprise ID を使用してログイン]**&#x200B;をクリックして、会社のシングルサインオンページに移動します。 Enterprise IDについて詳しくは、[会社または学校のアカウントにログイン ](https://helpx.adobe.com/jp/enterprise/kb/enterprise-id-faq.html#whatis)を参照してください。

![Adobe CX Enterpriseのサインインウィンドウに表示されているスクリーンショットに、Adobe IDの有無に関わらずサインインするためのオプションが表示されている](assets/adobe_id_login.png)

>[!NOTE]
>
>CX Enterprise ログインはセッションベースで、トークンは30日後に期限切れになります。

## Report Builder へのログイン

Report Builderにログインするには

1. Excel で、「**[!UICONTROL アドイン]**」をクリックします。
1. 「**[!UICONTROL ログイン]**」をクリックします。 次の方法を使用して署名することもできます。

   * 「**[!UICONTROL 作成]**」をクリックします。
   * リクエストマネージャーでリクエストを選択し、**[!UICONTROL 追加]**&#x200B;または&#x200B;**[!UICONTROL 管理]**&#x200B;をクリックします。 詳しくは、[ リクエストマネージャーでリクエストを選択](/help/analyze/legacy-report-builder/manage-requests/r-arb-manage-requests.md)を参照してください
   * Excelでリクエストをダブルクリックします。

1. 「[!UICONTROL ログイン]」ページのフィールドにデータを入力し、「**[!UICONTROL OK]**」をクリックします。
