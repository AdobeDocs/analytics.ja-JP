# 保存済みフィルター

{{legacy-arb}}

Report Builderでは、フィルターやその他のパラメーターを保存し、他のワークシートやワークブックで使用できます。 これらのパラメーターはAnalyticsに保存され、他のコンピューター上の他のReport Builder ユーザーが使用できるようにします。

フィルターの作成時に、[!UICONTROL  ページを選択] フォームにフィルターを保存します。 この手順の例については、[特定のフィルター](/help/analyze/legacy-report-builder/layout/c-filter-dimensions/t-specific-filters.md)を参照してください。
