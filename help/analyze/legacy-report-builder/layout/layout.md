# レイアウト - リクエストウィザード ステップ 2

{{legacy-arb}}

[!UICONTROL  リクエストウィザード：ステップ 2] ウィンドウでは、レポートレイアウト、ヘッダーの書式設定、およびディメンションのフィルタリングを出力用に設定できます。 ピボットレイアウトを使用すると、Excel でピボットテーブルを作成するのと似た方法でリクエストを設定できます。

また、「[カスタムレイアウト](/help/analyze/legacy-report-builder/layout/configure-the-custom-layout.md)」を使用してレポートデータを表示するセルを手動で選択することができます。
