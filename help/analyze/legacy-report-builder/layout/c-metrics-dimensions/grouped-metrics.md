# 同じリクエストでのトラフィック指標とコマース指標の混在

{{legacy-arb}}

個別のリクエストを作成する代わりに、1つのリクエストでトラフィック指標タイプとコマース指標タイプを組み合わせることができます。

トラフィックとコマースの指標は、リクエストウィザードの同じ指標カテゴリに表示されます。ステップ 2. グループ化されたカテゴリには、**[!UICONTROL 標準]**&#x200B;というラベルが付いています。
