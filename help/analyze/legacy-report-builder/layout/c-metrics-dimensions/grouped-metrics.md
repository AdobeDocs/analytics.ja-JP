# 同じリクエストでのトラフィック指標とコマース指標の混在

別々のリクエストを作成する代わりに、1 つのリクエストでトラフィック指標タイプとコマース指標タイプを組み合わせることができます。

トラフィック指標とコマース指標は、リクエストウィザード：手順 2 で、同じ指標カテゴリに表示されます。 グループ化されたカテゴリには、「**[!UICONTROL 標準]** というラベルが付けられます。
