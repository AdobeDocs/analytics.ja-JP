# 出力マッピングの削除

{{legacy-arb}}

ワークブック編集中に新しいリクエストの領域を確保したい場合、既存リクエストの削除が必要になることがあります。

スペースを確保する唯一の方法は、セルにマッピングされたリクエストを削除することです。 行または列を削除してスプレッドシート内のリクエストを削除すると、基になるリクエストの有効性が失われ（リフレッシュ時に失敗します）、 Excel メニューの選択&#x200B;**[!UICONTROL 編集]** > **[!UICONTROL 内容をクリア]**&#x200B;すると、セルに表示される値が削除されます。 セルの内容を更新することで、内容を復元できます。

スプレッドシートのセルの特定のマッピングを削除するには、行、列、または指標の項目を見つけて、**[!UICONTROL 削除]**&#x200B;をクリックします。
