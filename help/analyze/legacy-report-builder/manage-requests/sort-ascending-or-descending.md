# 昇順または降順の並べ替え

{{legacy-arb}}

リクエストマネージャーに表示されるリクエストを昇順または降順で並べ替えることができます。

表ヘッダーの列名をクリックします。 列には、テーブルヘッダーに三角形のアイコンが表示され、並べ替え順序が表示されます。 三角形が下を向いている場合、並べ替え順序は降順（英数字データの場合は9-1とZ-A）、上を向いている場合、並べ替え順序は昇順（英数字データの場合は1-9とA-Z）です。

デフォルトの並べ替え順は昇順です。
