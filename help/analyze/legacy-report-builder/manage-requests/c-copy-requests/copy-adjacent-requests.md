# 隣接するリクエストのコピー

{{legacy-arb}}

リクエストをコピーしてペーストするのと同じように、ショートカットメニューから「[!UICONTROL  リクエストをカット ]」を選択して、スプレッドシートの別の部分にリクエストを再配置することもできます。

リクエストを切り取ると元の場所からリクエストが削除されて、「[!UICONTROL リクエストを貼り付け]」を選択すると、新しい場所に配置されます。 特定のリクエストをカットした後に気が変わり、別のセルから別のリクエストをコピーまたはカットする場合、Report Builderは最初のリクエストを元のセルに残し、2番目のリクエストのみを実行（コピーまたはカット）します。

>[!NOTE]
>
>レポートビルダーでは、リクエストの切り取りまたは貼り付けのExcel取り消しコマンドはサポートされていません。

ブックの同じシートにコピー&amp;ペーストするだけではありません。 1つのシート内のリクエストをコピーし、同じブック内の別のシート内の場所に貼り付けることができます。

リクエストを一度に1つずつコピー&amp;ペーストするだけではありません。 スプレッドシートで複数のリクエストを選択し、スプレッドシートの空の領域に貼り付けることができます。 1つのリクエストをコピーしてペーストする場合と同様に、ペースト領域に、ペースト操作で置き換えられるリクエストを含むセルがないことを確認します。 ターゲットペースト領域に既に1つ以上のリクエストが含まれていることが判明した場合、Report Builderは、コピーまたはカットされたリクエストに対して「[!UICONTROL  リクエストをペースト ]」メニューを表示しません。 貼り付け操作の宛先として別のセルを選択して、リクエストが重複しないようにします。
