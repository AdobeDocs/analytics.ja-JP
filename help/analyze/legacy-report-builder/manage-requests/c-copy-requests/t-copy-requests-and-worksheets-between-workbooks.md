# ワークブック間でのリクエストおよびワークシートのコピー

{{legacy-arb}}

ソース ワークブック内のスプレッドシート全体を、1つまたは複数のターゲット ワークブック内のスプレッドシートにコピーします。 これを行うには、同じExcel インスタンスで少なくとも2つのワークブックを開く必要があります。

* 最初のソースワークブックには、セルにマッピングされたリクエストを含むスプレッドシート（ワークシート）が含まれています。

* 追加のターゲットワークブックは宛先です。 リクエストを含むスプレッドシートを貼り付ける前に、新しいターゲットワークブックごとに、ソースワークブックと同じレポートスイートにログインする必要があります。

>[!NOTE]
>
>ソースワークブックと同じレポートスイートを使用して、ターゲットワークブックにログインする必要があります。 両方のワークブックのリクエストは、同じレポートスイートログインを使用して作成する必要があります。

1. コピー元のワークブックのスプレッドシートを右クリックして、「**[!UICONTROL リクエストを含むワークシートをコピー]**」を選択します。
1. コピー先のワークブックで、スプレッドシートを右クリックして、「**[!UICONTROL リクエストを含むワークシートを貼り付け]**」を選択します。

   Excel の同じインスタンスを使用するというのは、そのコンピューター上で Excel プロセス（[!DNL excel.exe]）が 1 つだけ動作している状態にするということです。 2つのExcel インスタンスを起動し、1つ目のExcel インスタンスのブックから2つ目のExcel インスタンスのブックにワークシートをコピーしようとすると、Report Builderには、2つ目のExcel インスタンスのショートカットメニューにワークシートを貼り付けるオプションは表示されません。

   異なるレポートスイートを使用してソースブックとターゲットブックにログインした場合、貼り付け操作で表示される結果は、ターゲットブックの書式設定に影響を与える結果のみです。 指定されたレポートスイート（ソースワークブック）から取得されたリクエストの情報が、ターゲットワークブックで利用できないことを示すメッセージがReport Builderに表示されます。 ターゲット ワークブックに貼り付けられたリクエストを表示するには、ソース ワークブックと同じレポートスイートを使用してターゲット ワークブックにログインする必要があります。

   1つのワークブックのスプレッドシートから1つ以上のリクエストを、2つ目のワークブックが同じExcel インスタンスの別のドキュメントとして開いている限り、別のワークブックのスプレッドシートにコピーして貼り付けることができます。 両方のワークブックのリクエストは、同じレポートスイートログインを使用して作成する必要があります。
