# ワークシートのコピー

{{legacy-arb}}

同じワークブック内でスプレッドシートを複製します。

1. スプレッドシート内のセルの任意の箇所を右クリックして、「**[!UICONTROL リクエストを含むワークシートをコピー]**」を選択します。
1. 再度右クリックして、「**[!UICONTROL リクエストを含むワークシートを貼り付け]**」を選択します。

   新しいワークシートがワークブックに追加されます。 デフォルトでは、ワークシートは元のワークシートと同じ名前ですが、接尾辞（2）があります。 貼り付け操作を繰り返す場合、新しいワークシートには接尾辞（3）が付きます。 コピーしたスプレッドシートを貼り付けると、新しいスプレッドシートはアクティブなワークシートの右側（直後）に配置されます。

   リクエストのコピーとペーストとは異なり、Report Builderは、スプレッドシートをコピーする際に、直接入力されたセルの内容（列見出し、行、フォーマット用のラベルなど）をコピーしてペーストします。
