# 診断メッセージ

{{legacy-arb}}

診断メッセージによって、更新したリクエストの状況に関する情報を得られます。

![&#x200B; リクエストが正常に更新されたことを示す緑色のチェックマークのアイコン。](assets/icon_notice_success.gif) このブックを開いてからリクエストが正常に更新されたことを示します。 「診断」列には、リクエストで使用される時間、リクエストをマッピングする時間、リクエストが以前にキャッシュされた情報を使用するかどうかなど、リクエストに関するその他の情報が表示されます。

![要求の更新に失敗したことを示す感嘆符が付いた赤い三角形のアイコン。](assets/icon_notice_warn.gif) このアラートは、リクエストが正常に更新されなかった場合に表示されます。 エラーは、障害の原因として考えられる原因を示す診断列に記録されます。 選択した障害メッセージに関するヘルプについては、[&#x200B; トラブルシューティングの節](/help/analyze/legacy-report-builder/troubleshoot.md)を参照してください。
