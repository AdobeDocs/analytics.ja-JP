# すべてのシートのリクエストの更新

{{legacy-arb}}

すべてのシート上のリクエストを一括で更新することができます。

すべてのリクエストを更新するには、すべてのリクエストで使用されているすべてのレポートスイートにアクセスできる必要があります。 使用されているすべてのレポートスイートのサブセットのみにアクセス権を付与する資格情報を使用すると、ログイン時に、一部のリクエストを更新できないことを示す警告メッセージが表示されます。 この場合、すべてのリクエストを更新すると、アクセス権のないリクエストは更新されません。

複数のリクエストを更新すると、リクエストが元のリクエストであるか依存するリクエストであるかに基づいて、システムはリクエストを収集し、優先順位を付けます。 更新の進行状況に関するメッセージが表示されます。 更新プロセスは、要求するデータ量に応じて、最大で数分かかる場合があります。

すべてのシートのリクエストを更新するには

1. 「**[!UICONTROL 管理]**」をクリックします。
1. 「**[!UICONTROL すべてのシート]**」を有効にします。
1. 「**[!UICONTROL 更新]**」をクリックします。
1. 日付参照を指定して「**[!UICONTROL OK]**」をクリックします。
