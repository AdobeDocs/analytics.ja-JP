# 分類

{{legacy-arb}}

分類の分類は、分析レポート データを関連するプロパティにマッピングするために使用されます。 分類はさまざまな目的で使用できますが、キャンペーンのトラッキングコード（内部および外部）、および製品IDの分類に最もよく使用されます。

例えば、[!UICONTROL Campaign] レポートには、[!UICONTROL Manager]分類と[!UICONTROL Outlet]分類が含まれる場合があります。 これらの分類は、特定の階層に従って並べ替えることができ、レポート値の各グループをサブグループに分けることができます。

分類には、レポートスイートの他のレポートにアクセスするのと同じ方法でアクセスし、ディメンションはグループ内の最上位レポートとして表示されます。 レポートに分類がある場合は、メニューサブグループが作成され、レポートとそのすべての分類が同じ場所に収集されます。 レポートは、分類のリストの最後のエントリとして表示されます。

[分類](/help/components/classifications/classifications-overview.md)を参照してください。
