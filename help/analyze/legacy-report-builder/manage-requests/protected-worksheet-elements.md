# 保護されたワークシートの要素

{{legacy-arb}}

ワークシートを保護するためにセルをロックすると、リクエストの作成、編集、更新ができません。 リクエストマネージャーを使用する前に、ワークシートからすべての保護を削除します。

要求を更新し、更新要求プロセス中にセルを編集しようとすると、スプレッドシートが現在保護されていることを示すExcel トーストが表示されます。 更新リクエストプロセスが完了すると、スプレッドシートにアクセスし、一時的なスプレッドシート保護が削除されます。
