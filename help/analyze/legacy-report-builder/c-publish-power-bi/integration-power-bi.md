# Power BI 用 Adobe Analytics コンテンツパック

{{legacy-arb}}

Adobe Analytics コンテンツパックは、サイトトラフィックとユーザーディメンションに関するインサイトを提供する、事前定義済みのPower BI ダッシュボードと一連のPower BI レポートを提供します。 提供されるダッシュボードやレポートを使用したり、最も関心のある情報をハイライト表示するためにカスタマイズしたりできます。 データは 1 日に 1 回自動的に更新されます。

次の 3 つのコンテンツパックが利用できます。

* Adobe Analytics
* Adobe Analytics モバイルアプリ分析
* Adobe Analyticsのトラフィック分析

詳しくは、[Power BI 用 Adobe Analytics コンテンツパック](https://powerbi.microsoft.com/en-us/documentation/powerbi-content-pack-adobe-analytics/)を参照してください。
