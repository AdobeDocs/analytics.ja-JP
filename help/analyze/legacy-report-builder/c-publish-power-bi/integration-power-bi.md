# Power BI 用 Adobe Analytics コンテンツパック

{{legacy-arb}}

Adobe Analyticsコンテンツパックには、サイトのトラフィックとユーザーディメンションに関するインサイトを提供する、事前定義済みのPower BIダッシュボードと一連のPower BIレポートが用意されています。 提供されるダッシュボードやレポートを使用するか、それらをカスタマイズして、最も関心のある情報を強調表示できます。 データは 1 日に 1 回自動的に更新されます。

次の 3 つのコンテンツパックが利用できます。

* Adobe Analytics
* Adobe Analytics Mobile App Analytics
* Adobe Analytics Traffic Analysis

詳しくは、[Power BI 用 Adobe Analytics コンテンツパック](https://powerbi.microsoft.com/en-us/documentation/powerbi-content-pack-adobe-analytics/)を参照してください。
