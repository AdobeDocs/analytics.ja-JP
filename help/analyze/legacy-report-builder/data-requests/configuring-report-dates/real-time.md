# リアルタイムレポート

{{legacy-arb}}

Web ページトラフィックをリアルタイムで表示し、ページビュー数をランク付けします。これにより、サイトのトレンドをよりすばやく把握できるようになります。

Adobe Analyticsでは、リアルタイムのレポートのために、高頻度の指標とサイト分析を使用して、動的なニュースや小売web サイトのトラフィックとページビューのトレンドを視覚的にレポートします。 Adobe Real-Time CDPは、データを収集してから数秒で、分単位でトレンドを把握できます。 データを収集し、インターフェイスにストリーミングします。これにより、コンテンツとコンバージョンのリアルタイムの相関関係と追跡が可能になります。

次のことができます。

* 既存の指標、ディメンション、分類を使用して、レポートスイートごとに最大3つのリアルタイムレポートを作成します。 セカンダリディメンションを使用して、プライマリディメンションに関連付ける（または分類する）ことができます。
* サイト全体の指標に加えて、レポートごとに3つのディメンション（または分類）を追加します（1つのプライマリと2つのセカンダリ）。
* カスタムイベント、ショッピングカートイベント、インスタンスを使用できます。
* 最大20時間の履歴データをリアルタイムで表示。

ユーザーとグループの権限をリアルタイムで尊重。 例えば、収益を表示する権限がない場合、収益データを含むリアルタイムのレポートを表示することはできません。 eVar （コンバージョン指標）はサポートされていません。 リアルタイム権限は、[管理ツール ](/help/admin/tools/manage-rs/edit-settings/realtime/t-realtime-admin.md)で有効になっています。

>[!NOTE]
>
>[!UICONTROL  リアルタイムレポート ]は、以前にリリースされた[現在のデータを含める](/help/analyze/legacy-report-builder/options.md) （データ最新性）機能と同じではありません。これは、最終化される前にデータを表示することで、標準レポートの遅延を減らしました。
