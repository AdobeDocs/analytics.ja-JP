# リクエストウィザードの定義 - 固定日付

{{legacy-arb}}

Report Builder における固定日付の定義です。

| フィールド | 定義 |
|--- |--- |
| このレポートスイート | 保存された日付範囲を現在のレポートスイートでのみ使用できるようにします。 |
| すべてのレポートスイート | 保存された日付範囲を、ログインユーザーがアクセスできるすべてのレポートスイートで使用できるようにします。 |
| パラメーター名を入力 | 保存した日付の名前を入力します。 この日付は、Report Builderのテンプレートとして使用できます。 この名前は、保存済みの日付を適用できる場所に表示されます。 テンプレートの名前を変更しても、以前に作成したデータリクエストで使用される日付設定には影響しません。 |
