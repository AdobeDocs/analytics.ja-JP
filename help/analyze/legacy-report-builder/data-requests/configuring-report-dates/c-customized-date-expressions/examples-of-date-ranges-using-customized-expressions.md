# カスタマイズされた式を使用した日付範囲の例

{{legacy-arb}}

カスタマイズされた式で日付範囲を使用する場合の例、注意事項および構文に関する注意点について説明します。

この表は、今日の日付がグレゴリオ暦を使用した2011年11月10日月曜日であると仮定しています。

| 例 | 日付範囲 | 式のカスタマイズ | レポートの日付範囲 |
|---|---|---|---|
|  | | **送信元** | **設定値** |
| 1 | 2週間前 | `cw-2w  \| cw-1w-1d` | 10月26日～11月1日 |
| 2 | 昨年の5か月目の最初の3日間 | `cy-1y+4m  \| cy-1y+4m+2d` | 2010年5月1日～5月3日 |
| 3 | 4週間前から1週間 | `cw-4w  \| cw-3w-1d` | 10月12日～10月18日 |
| 4 | 前年の先週 | `cw-53w  \| cw-52w-1d` | 2010年11月9日（PT） |
| 5 | 2か月前から1か月 | `cm-2m  \| cm-1m-1d` | 9月1日～30日 |
| 6 | 12か月前の前年 | `cm-12m  \| cm-11m-1d` | 2010年11月1日～11月30日 |

## 例に関する注意事項 {#section_37801B0D6D364ABAA8DCE3A4C0123B2C}

**例 1**

今日が2011年11月10日の月曜日の場合は、現在の日付を取得し、10月の最後の週を取得するために1週間を減算します。

**例 2**

年の初めに4か月（1月の月）を追加して5月の月を取得します。月の最初の日に2日を追加して、月の3日目を取得します。

## 構文メモ {#section_555D6563B2D94FA3BDD801DC0B8C289D}

ほとんどの日付範囲をカバーするカスタマイズされた式は、2つの用語を演算子にリンクすることによって作成できます。 項は、整数乗数とピリオドの略語の組み合わせです。 用語の例は18dです。 演算子の例は+です。

* 演算子と項の間には空白は使用できません。
* 次の略語のみを使用します：cd cw cm cq cy d w m q y
* ベストプラクティスは、開始日と終了日で同じ日付参照を使用することです（cd、cd、またはcw、cw、またはcy、cy）。 日付の参照を混在すると、年の特定の時点で日付が無効になる場合があります。
* 略語d w m q yの有効な倍数は、53d 3w 5q 9m 2yのように、略語の先頭に整数（1 2 3 ...）を付けて形成します
* 非整数は使用できません。
* 省略形の先頭にゼロのみを付けないでください。 例えば、0wは許可されていません。
* 省略形を連結するには、次の演算子を使用します。+ -
* 日付範囲は現在の期間に対して考慮する必要があるので、式の最初の用語は常にcで始まります。
