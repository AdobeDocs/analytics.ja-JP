# セルからの日付の指定

{{legacy-arb}}

日付範囲を指定するには、リクエストを含むワークシートからセルを選択します。 レポートビルダーは、これらのリクエストで特定の日付範囲情報を使用します。 今日の日付を選択すると、リクエストが実行された時刻に基づく部分的なデータが表示されます。

**セルから日付を設定するには**

1. [!UICONTROL リクエストウィザード：ステップ 1] で、「**[!UICONTROL セルから日付を選択]**」を選択します。
1. **[!UICONTROL From]**&#x200B;および&#x200B;**[!UICONTROL To]** フィールドにセル参照を入力するか、セレクターをクリックして、開始日と終了日を含むリクエストを含むセルを選択します。

   例えば、日付範囲を「昨日」に設定した Report Builder リクエストを作成し、「today()-1」と同じセルにリクエスト日を出力します。
