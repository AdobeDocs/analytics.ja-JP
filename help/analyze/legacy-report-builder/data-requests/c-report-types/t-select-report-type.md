# レポートタイプの選択

{{legacy-arb}}

データリクエストのベースレポートタイプを選択する手順です。

1. [!UICONTROL リクエストウィザード：ステップ 1] ウィンドウを開きます。
1. **[!UICONTROL レポートスイート]**&#x200B;メニューからレポートスイートを選択します。
1. データリクエストのベースレポートタイプをクリックして選択します。
