# ブックマークされたレポートおよびダッシュボードレポートのインポート

{{legacy-arb}}

ブックマーク済みのすべてのレポートとダッシュボードレポートは、リクエストウィザードの手順1にディメンションとして一覧表示され、Report Builder リクエストとして読み込むことができます。

ブックマーク付きレポートを選択すると、リクエストウィザードによって、このブックマーク付きレポートを定義するすべてのディメンションと指標が入力されます。 選択したブックマークに基づいて、日付範囲、粒度、選択したセグメントも更新されます。

次に、リクエストウィザードの手順1でダッシュボードとそのレポートレットを表示する方法を示します。

![&#x200B; リクエストウィザードを示すスクリーンショット ステップ 1/2 ダッシュボードの取得とブックマークの取得を強調表示します。](assets/import_dashboard_reportlet.png)

**[!UICONTROL ダッシュボードの取得]**&#x200B;または&#x200B;**[!UICONTROL ブックマークの取得]**&#x200B;をクリックすると、既存のダッシュボードやブックマークのデータが取得され、ワークシートに貼り付けられます。

>[!NOTE]
>
>データのみがインポートされるので、ブックマークにチャートが含まれている場合や、ダッシュボードレポートレットがチャートのみで構成されている場合は、チャートの作成に使用されているデータのみがインポートされます。

ダッシュボードレポートレット（またはブックマーク）を読み込んでリクエストを作成すると、そのリクエストはレポートレット（またはブックマーク）のプライマリディメンションに関連付けられます。 その結果、リクエストを編集すると、ツリービューでダッシュボードレポートレットのツリービューノード（またはブックマークノード）が選択されなくなります。代わりに、プライマリディメンションが選択されます。
