# Activity Map の実装

Activity Map は、Adobe Analytics の Adobe Experience Platform Launch 拡張機能の[リンクトラッキング](https://experienceleague.adobe.com/docs/launch/using/extensions-ref/adobe-extension/analytics-extension/overview.html?lang=ja#link-tracking)の下にあるチェックボックスを使用して実装できます。
