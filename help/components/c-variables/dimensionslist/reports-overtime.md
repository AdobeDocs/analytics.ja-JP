# 長期間

購入、買い物かごおよびカスタムイベントをレポートするセクションで使用されます。ページビュー数に類似しており、これらのレポートには、1 日、1 週間、1 ヶ月などの特定の期間に収集された、単一の成功指標によるデータが表示されます。
