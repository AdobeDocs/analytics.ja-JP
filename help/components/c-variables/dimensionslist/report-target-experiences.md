# ターゲットエクスペリエンス

Target で設定したエクスペリエンス（レシピ）のリストを表示します。Analytics for Target 統合が有効になっている必要があります。

エクスペリエンスは、Adobe Target で mbox にどのコンテンツを表示するかを定義するもので、「レシピ」とも呼ばれます。
