# 初回購入までの日数

顧客がサイトを初めて訪問してから最終的に購入するまでの日数を示します。

例えば、訪問者が最初の訪問の 1 日後に買い物をした場合、訪問者の cookie が有効である間は、訪問やイベントはレポートの&#x200B;*「1 日」*&#x200B;行項目に表示されます。
