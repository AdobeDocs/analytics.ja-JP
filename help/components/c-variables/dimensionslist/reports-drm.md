# デジタル著作権管理（DRM）

サポートされるデジタル著作権管理（DRM）のタイプ別にモバイルデバイスをグループ分けします。DRM グループには、「転送禁止（Forward Lock）」、「結合配信（Combined Delivery）」、「分離配信（Separate Delivery）」、および「不明」があります。
