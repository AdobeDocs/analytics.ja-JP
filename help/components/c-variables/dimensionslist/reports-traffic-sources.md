# トラフィックソース

サイトにアクセスする際に経由している Web 上の場所についての情報が表示されます。どの検索エンジンまたはドメイン外の Web サイトから訪問者が送り込まれているかがわかります。

## トラフィックソース {#topic_6718F8EFD5984DC5839B9E8F6CC45EDA}

サイトにアクセスする際に経由している Web 上の場所についての情報が表示されます。どの検索エンジンまたはドメイン外の Web サイトから訪問者が送り込まれているかがわかります。

このメニューのレポートは、次の 3 つの基本カテゴリに分類できます。

* 検索エンジン
* 検索キーワード
* リファラーと参照ドメイン
