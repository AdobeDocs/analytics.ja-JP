# JavaScript のバージョン

サポートされる Java バージョン別にモバイルデバイスをグループ分けします。JavaScript のバージョンレポートでは、デバイスでサポートされている Java 仕様要求（JSR）により Java バージョンを識別します。

JSR の詳細については、[Java Community Process](https://jcp.org/en/jsr/overview) の Web サイトをご覧ください。
