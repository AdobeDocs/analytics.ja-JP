# キャンペーン

広告活動の効果についての情報が表示されます。どのような広告活動が最も多くのトラフィックを獲得したか、どの従業員がその活動を担当したかを調べることができます。

これらのレポートは通常カスタマイズされるので、分析ごとに異なります。
