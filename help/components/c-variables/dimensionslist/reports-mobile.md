# モバイル

モバイルデバイスからのWebサイトへのアクセスに関する情報を表示します。 モバイルレポートは、訪問者の識別に使用されるモバイルデバイスに関するデータを収集することで、表示の識別を改善します。 モバイルレポートは、デバイスの製造元、および画面サイズ、ビデオ、オーディオ、cookieのサポート、その他の指標などの機能を識別できます。 モバイルレポートは、すべてのレポートスイートで使用できます。

## モバイル {#topic_D306EA4558194488AC47A45B9C570150}

モバイルデバイスからのWebサイトへのアクセスに関する情報を表示します。 モバイルレポートは、訪問者の識別に使用されるモバイルデバイスに関するデータを収集することで、表示の識別を改善します。 モバイルレポートは、デバイスの製造元、および画面サイズ、ビデオ、オーディオ、cookieのサポート、その他の指標などの機能を識別できます。 モバイルレポートは、すべてのレポートスイートで使用できます。

>[!NOTE]モバイルレポートにはモバイル Web トラフィックのみが表示されます。ネイティブのiOS、Androidおよびその他のSDKを使用して収集されたモバイルアプリデータは、モバイルアプリレポートに表示され、管理ツールで有効になっている場合にのみ表示されます。
