# 注文件数

指定期間内の Web サイトでの注文件数を表示します。他の指標によって個々の期間を細分し、その時間枠内で最も注文件数に寄与した項目（製品やキャンペーンなど）を示すことができます。

| 使用 | 説明 |
|---|---|
| トラフィック | 該当なし |
| コンバージョン | *`Order`*&#x200B;は、購入イベントが設定された回数です。 |
| Ad Hoc Analysis | *`Order`*&#x200B;は、購入イベントが設定された回数です。 |
