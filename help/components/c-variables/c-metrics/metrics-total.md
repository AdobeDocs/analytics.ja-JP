# 合計

レポート期間内の全レポート行項目の値。現在フィルターが選択されている場合、合計はレポートスイートの合計ではなく、フィルター処理された合計を表している可能性があります。フィルターが選択されていない場合は、合計はレポートスイートの合計を表します。
