# リロード回数

同じページ名が 2 回連続で読み込まれたときにカウントされます。これは、一般に、そのページが更新されたことを示します。

同じ訪問で 2 回同じページを訪れても、連続して訪れないかぎりはリロードとしてカウントされません。
