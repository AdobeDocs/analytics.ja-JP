# 売上高

購入イベントで売上高が取り込まれ、各商品の注文の総額として定義されます。この値は購入イベントから取得されます。

| 使用 | 説明 |
|---|---|
| トラフィック | 該当なし |
| コンバージョン | [!UICONTROL 購入]イベントで売上高が取り込まれ、その注文の総額と各商品の総額として定義されます。 |
| Ad Hoc Analysis | [!UICONTROL 購入]イベントで売上高が取り込まれ、その注文の総額と各商品の総額として定義されます。 |
