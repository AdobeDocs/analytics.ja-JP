# [!UICONTROL Content Velocity]

[!UICONTROL Content Velocity] は、標準のAdobe Analytics指標テンプレートです。 [!UICONTROL Content Velocity] は、[[!UICONTROL Page Views] | [!UICONTROL Visit Participation]] / [[!UICONTROL Visits]]を使用して、特定のコンテンツ要素（ページ、サイトセクションなど）に与える影響を測定できます。がダウンストリームコンテンツに含まれていた問題を修正しました。 Webサイトやモバイルアプリでユーザーを保持しているコンテンツと保持していないコンテンツを把握するのに役立ちます。

![](assets/cont-velo-1.png)

[!UICONTROL Content Velocity] は、検索またはタグ(#Adobe Template)を使用して左側のレールをフィルタリングすることで、Adobe Workspaceで見つかります。

![](assets/cont-velo-2.png)
