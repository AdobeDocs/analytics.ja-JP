# チェックアウト

「チェックアウト」指標[指標](overview.md)には、訪問者がチェックアウトプロセスを開始した回数が表示されます。 この指標は、顧客が買い物かごに商品を入れて購入する準備ができているコンバージョンファネルの部分を理解したい場合に役立ちます。

## この指標の計算方法

この指標は、`scCheckout` が [`events`](/help/implement/vars/page-vars/events/events-overview.md) 変数に存在するヒット数をカウントします。
