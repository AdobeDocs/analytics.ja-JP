# 買い物かごへの追加

「買い物かごへの追加件数」指標は、訪問者が買い物かごに何かを追加した回数を示します。この指標は、顧客が商品に十分興味を持って買い物かごに商品を追加するコンバージョンファネルの部分を理解したい場合に役立ちます。

## この指標の計算方法

この指標は、`scAdd` が [`events`](/help/implement/vars/page-vars/events/events-overview.md) 変数に存在するヒット数をカウントします。
