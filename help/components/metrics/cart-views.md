# 買い物かご表示

「買い物かご表示」指標は、訪問者が買い物かごを表示した回数を示します。この指標は、顧客が買い物かごのコンテンツを表示したコンバージョンファネルの部分を理解したい場合に役立ちます。

## この指標の計算方法

この指標は、`scView` が [`events`](/help/implement/vars/page-vars/events/events-overview.md) 変数に存在するヒット数をカウントします。
