# リロード回数

「リロード回数」指標は、再読み込み中にディメンション項目が存在した回数を示します。 訪問者がブラウザーを更新することは、再読み込みをトリガーする最も一般的な方法です。

## この指標の計算方法

この指標は、ディメンションに前のヒットと同じ値が含まれるヒット数をカウントし [`Page`](../dimensions/page.md) ます。
