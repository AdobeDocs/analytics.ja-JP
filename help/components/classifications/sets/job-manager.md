# 分類セットジョブマネージャ

分類セットジョブマネージャーを使用すると、分類セットから生成された、現在の分類ジョブと完了した分類ジョブを表示できます。 また、このインターフェイスを使用して、特定のジョブの分類をダウンロードしたり、追加のデータをジョブにアップロードしたりできます。
