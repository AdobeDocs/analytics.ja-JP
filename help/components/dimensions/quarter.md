# 四半期

「四半期」 [ディメンション](overview.md) は、特定の指標が発生した四半期をレポートします。 最初のディメンション項目は日付範囲の最初の四半期で、最後のディメンション項目は日付範囲の最後の四半期です。このディメンションは、経時的に指標を表示できるトレンドレポートに最適です。

## このディメンションへのデータ入力

このディメンションは、すべての実装において初期設定の状態で動作します。レポートスイートにデータが含まれていれば、このディメンションは機能します。

## ディメンション項目

ディメンション項目には、指定した日付の 3 ヶ月の四半期と年が含まれます。
