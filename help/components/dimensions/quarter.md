# 四半期

「四半期」ディメンションは、特定の指標が発生した四半期をレポートします。 最初のディメンション値は日付範囲の最初の四半期で、最後のディメンション値は日付範囲の最後の四半期です。 このディメンションは、経時的に指標を表示できるトレンドレポートに最適です。

## このディメンションにデータを入力する

このディメンションは、すべての実装で初期設定の状態で動作します。 レポートスイートにデータが含まれる場合、このディメンションは機能します。

## 分析コード値

ディメンション値には、指定した日付の3か月の四半期と年が含まれます。
