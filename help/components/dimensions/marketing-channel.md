# マーケティングチャネル

*このヘルプページは、分析ワークスペースのマーケティングチャネルディメンションを参照します。 マーケティングチャネルがコンセプトとしてどのように機能するかについて詳しくは、「[マーケティングチャネルの概要](../c-marketing-channels/c-getting-started-mchannel.md)」を参照してください。*

「マーケティングチャネル」ディメンションは、「ラストタッチチャネル」ディメンションと同じよう [に機能](last-touch-channel.md) します。 このディメンションは、カスタムアトリビューションを使用するレポートで使用するために導入されました。 カスタムアトリビューションマーケティングチャネルレポートを表示したユーザーは、「ラストタッチ」とカスタムアトリビューションが正確でないので、「ラストタッチ」ラベルと混同されていました。 アドビでは、汎用の「マーケティングチャネル」ディメンションを導入したので、アナリストは、誤った「ラストタッチ」ディメンションラベルを使用することなく、カスタムアトリビューションマーケティングチャネルレポートを作成できます。
