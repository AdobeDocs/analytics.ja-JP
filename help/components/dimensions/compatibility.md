# Analytics のディメンションの互換性

このページには次のリストがあります [寸法](overview.md) 対応する Analytics 機能でサポートされます。

>[!NOTE]
>
>カスタム変数名、分類、訪問者属性は、このリストには含まれません。これらのディメンション項目は、個々のレポートスイートに固有の値です。
