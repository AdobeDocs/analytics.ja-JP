# 主要なメリット{#key-benefits}

この統合には、次のようなメリットがあります。

* 電子メールマーケティングおよび解析データを1つのレポートインターフェイスに統合する
* コンバージョンおよびサイトの成功に対するコンバージョンと貢献度によって電子メールキャンペーンを最適化する
* 主要訪問者に対して再マーケティングし、動的マーケティングセグメントに基づいてセグメント化します
