# 主要なメリットと機能{#key-benefits-and-features}

この統合には、次のようなメリットがあります。

* 電子メールマーケティングおよび解析データを1つのレポートインターフェイスに統合できます。
* コンバージョンおよびサイトの成功に対する貢献度によって電子メールキャンペーンを最適化します。
* 主要訪問者に再マーケティングし、動的マーケティングセグメントに基づいてセグメント化します。
