# 統合の確認{#verifying-the-integration}

Lyris/Adobe Analyticsの統合が成功したことを確認する手順です。

すべてのデプロイメント手順が完了したら、統合のデータ転送が正常に完了したことを確認できます。

>[!NOTE]
>
>データ交換が開始されるまで数日かかります。統合をアクティベートした後、Lirisにお問い合わせください。
