# 指標で並べ替え

Data Warehouseで、指標の値が降順で並べ替えられたランク付きの内訳レポートを提供します。

指標別に並べ替えることで、Data Warehouse レポートの解釈が容易になり、これらのレポートを他のAnalyticsの分類レポートビューと比較しやすくなります。

次に、「指標の並べ替え」オプションを有効にすると、Data Warehouse レポートの行が並べ替えられる方法を示します。

日付の精度、レポートディメンションまたは指標の設定方法、「最大行」が設定されているかどうかに基づいて、Data Warehouse レポートを「指標の並べ替え」で整理する方法は4つあります。

* **レイアウト 1**：行項目が辞書順で並べ替えられます（デフォルト）。 「最大行」が設定されている場合、最初のN行のみがレポートに表示されます。
* **レイアウト 2**：Data Warehouse は、レポートのすべての行に対して指標の並べ替えを適用します。 最初の指標値の関連付けは、2番目の指標によって解除され、次に3番目の指標によって解除されます。 すべての指標が関連付けられると、分類ライン項目の標準ディクショナリ順序が適用されます。
* **レイアウト 3**: レイアウト 2として、上位N行（つまり、「最大行」で設定された数）のみがレポートに出力されます。
* **レイアウト 4**: レイアウト 2として。ただし、各日付粒度期間の行項目は、グループ化され、それぞれの時間範囲内で並べ替えられます。
