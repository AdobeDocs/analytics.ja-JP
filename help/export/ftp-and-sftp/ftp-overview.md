# Adobe CX EnterpriseでのFTPおよびSFTPの使用

FTP および SFTP を使用してコンピューターおよびサーバーの間でファイルを転送する方法について学習します。

Adobe ソリューションおよび製品でFTPを使用する方法：

* 大規模な分類データセットをAnalyticsにアップロードします。
* 顧客属性をCX Enterpriseにアップロードして、ソリューションで使用します。
* Analyticsを使用して、FTP ベースのデータソースを作成および管理します。このデータソースでは、FTP ファイル転送を利用して、オフラインまたは履歴データをCX Enterpriseに読み込みます

これらの記事では、各製品での FTP の使用手順については必ずしも説明しません。 代わりに、アドビで FTP を使用する場合のベストプラクティスについて説明します。
