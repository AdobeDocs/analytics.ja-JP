# FTPとSFTPの処理時間

分類やデータソースの FTP ファイルの処理時間は、既に処理キューに登録されているファイルの数、およびアップロードしたファイルのサイズに応じて異なります。

送信するファイルが5 MBを超える場合、読み込み時間が若干長くなる可能性があります。 1～2日後に完了したファイルがFTP サイトに表示されませんが、アップロードされたデータがレポートツールに反映されない場合は、組織のサポートされているユーザーがAdobe カスタマーケアに連絡して、ファイルに問題があるかどうかを確認する必要があります。

FTP ファイルを作成する際は、必ず通知受信者のメールアドレスを追加してください。 そうすることで、ほとんどの問題が受信者に通知を送信することになります。
