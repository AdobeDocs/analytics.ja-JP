# 削除された FTP データおよび FTP アカウントの復元

FTP アカウントから削除されたデータは、いかなる場合でもアドビで復元することはできません。

ご使用の予定の FTP アカウントが無効になっている場合は、担当のAdobeアカウントチームにお問い合わせください。担当のアカウントチームにお問い合わせください。
