# FTP の制限とデータ保持期間

アドビのすべての FTP アカウントでは、最大で 10 GB のデータを保存できます。これらの FTP アカウントでは、ファイルは 90 日間保持されます。

## 同時接続の制限

* ソース IP あたりのサーバーへの同時接続数は 50 以下
* ソース IP あたりのサーバーへの新しい接続数は 60 秒ごとに 20 以下
