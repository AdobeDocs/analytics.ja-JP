# FTPおよびSFTPの制限とデータ保持

すべてのAdobe FTPおよびSFTP アカウントには、データストレージの制限が100 GBです。 これらの FTP アカウントでは、ファイルは 90 日間保持されます。

## 同時接続の制限

* ソース IP あたりのサーバーへの同時接続数は 50 以下
* ソース IP あたりのサーバーへの新しい接続数は 60 秒ごとに 20 以下
