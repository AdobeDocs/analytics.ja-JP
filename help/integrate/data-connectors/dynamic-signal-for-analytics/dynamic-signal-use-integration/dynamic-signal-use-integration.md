# Using the Integration{#using-the-integration}

導入が完了すると、この統合によって提供される追加機能の使用を開始できます。

**注意**:24~48時間かかることがあります。これにより、Adobe Analyticsレポート内にある動的シグナルデータの一部が表示され始めます。

次のアクションによって、Adobe Analytics内のこの統合から値が追加されます。
